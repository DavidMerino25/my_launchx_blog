#Uso y significado de 5 palabras reservadas en Java ###boolean Define una variable booleana para los valores “true” o “false” solamente. Por defecto, el valor del tipo primitivo booleano es falso. Esta palabra clave también se usa para declarar que un método devuelve un valor del tipo primitivo booleano. Ejemplo:

java boolean bandera = true;
