La Cultura de Desarrollador de Software.

Un Desarrollador de Software debe tener siempre tener las herramientas para el desarrollo de software bien presentes y adaptarse siempre a los nuevos cambios. Debe de tener siempre la capacidad de llevar a cabo sus actividades con mucha seguridad y dar siempre lo mejor de si Siemore debe de conocer el proceso de desarrollo de software, trabajar en equipo, valores como la lealtad y la honestidad y responsabilidad y siempre tener como prioridad cumplir los objetivos del proyecto
